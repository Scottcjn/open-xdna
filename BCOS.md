## License boundary (important)

- **Original work** (scripts, hand-authored AIE kernels in `examples/kernels/*.cc`, docs,
  benchmark harnesses): **AGPL-3.0-or-later** — `SPDX-License-Identifier: AGPL-3.0-or-later`
  on every source. A **commercial / proprietary license** is available (`COMMERCIAL.md`) —
  valid because this is fully-owned original work.
- **AMD's XDNA/XRT driver and IRON/mlir-aie + Peano are NOT vendored.** They are built/installed
  from upstream (`amd/xdna-driver`, `Xilinx/mlir-aie`, `Xilinx/llvm-aie`), licensed separately
  under **Apache-2.0 WITH LLVM-exception**. No AMD code or NPU firmware is redistributed here.
  open-xdna *orchestrates* these tools; it does not derive from or embed them.

## What this repo proves (test evidence detail)

Verified on a real Ryzen 7 8845HS (XDNA1 / `RyzenAI-npu1`), Ubuntu 25.10 / kernel 6.17 / GCC 15:
- gen-1 NPU bring-up (open-source XRT + driver + firmware), DKMS-persistent
- `512³` int16 matmul (~68 GFLOPS) and a 2-layer MLP forward pass on the NPU
- hand-authored AIE collapse kernels (`collapse`, `collapse_rt`, `collapse_fused`), `aie::reverse`
  shuffle, cross-tile reduce, and dense stream compaction — each with a bit-exact self-check.
